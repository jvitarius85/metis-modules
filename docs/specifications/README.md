# Metis Governance Pack

This folder contains the authoritative governance documents for the Metis platform.

## Required Reading Order

1. `METIS_DEVELOPMENT_PLAYBOOK.md`
2. `METIS_ARCHITECTURE_CONTRACT.md`
3. `METIS_SECURITY_MODEL.md`
4. `METIS_SYSTEM_FLOW.md`
5. `METIS_CORE_SERVICES.md`
6. `METIS_MODULE_SPEC.md`
7. `METIS_MODULE_GENERATOR_CONTRACT.md`
8. `METIS_FRONTEND_ARCHITECTURE.md`
9. `METIS_UI_DESIGN_RULES.md`

All other documents extend and enforce these core rules.

## Compliance Enforcement

For module and security enforcement requirements, also review:

- `METIS_MODULE_SPEC.md`
- `METIS_MODULE_DEVELOPMENT_CONTRACT.md`
- `METIS_SECURITY_MODEL.md`
- `METIS_DEPLOYMENT_RULES.md`

These documents define hard module compliance validation, fail-isolated module boot behavior, required audit logging, and release/deployment compliance gates.
