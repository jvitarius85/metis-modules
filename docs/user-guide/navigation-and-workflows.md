# User Guide

## Navigating the Interface

- Use the sidebar to move between modules and sub-views.
- Use breadcrumbs on detail screens to move back to parent lists.
- Use the Help toggle or `Shift+H` to show contextual guidance.

## Managing Records

- Core list screens expose search, filtering, detail pages, and AJAX-backed updates.
- Inline edits, modals, and guided walkthroughs all rely on the shared Metis UI runtime.

## Reports

- Finance, donations, and portal dashboards aggregate activity instead of issuing repetitive row-by-row queries.
- Saved report definitions and export flows use module-specific AJAX endpoints and PDF/reporting services.
