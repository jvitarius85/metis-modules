# Portal

Open shared dashboards and common workspace tools.

## Routes

- Base route: `/portal`
- `/portal/dashboard` -> `dashboard.php`
- `/portal/email_usage` -> `email_usage.php`

## UI Components

- **Dashboard** template: `dashboard.php`
- **Email Usage** template: `email_usage.php`

## APIs

- No dedicated AJAX controller was discovered for this module.

## Database Tables Used

- `board_action_items` (`metis_board_action_items`)
- `board_meetings` (`metis_board_meetings`)

## Assets and Extension Hooks

- CSS: `portal.css`
- JS: `portal.js`
- Registered help topics: `portal.dashboard`, `portal.email_usage`
